# RentFlow Logo Files

## Files included

| File | Use case |
|---|---|
| `rentflow-logo-primary.svg` | Main logo — icon + wordmark on cream bg. Use on white or light surfaces. |
| `rentflow-logo-reversed.svg` | Logo on brand green (#0F4C3A). For headers, banners, email templates. |
| `rentflow-logo-dark.svg` | Logo on midnight green (#0A1F14). For dark-mode UI, splash screens. |
| `rentflow-logo-stacked.svg` | Stacked variant — icon above wordmark. Use when vertical space is available. |
| `rentflow-icon.svg` | Square icon mark only (200×200). For app icons, favicons, social media profile pictures. |
| `rentflow-icon-circle.svg` | Circular icon mark. For WhatsApp profile picture, social avatars. |
| `rentflow-icon-white.svg` | Icon mark on transparent/frosted bg. Drop onto any dark surface. |
| `rentflow-wordmark.svg` | Wordmark only, no icon. For co-branding or inline text contexts. |
| `rentflow-wordmark-white.svg` | White wordmark only. For dark backgrounds without the icon. |
| `rentflow-favicon-32.svg` | Optimised 32×32 favicon. Add to your HTML: `<link rel="icon" href="/rentflow-favicon-32.svg">` |

## Typography (wordmark)

- **"Rent"** — Helvetica Neue, weight 200 (ultralight), tracking −0.04em
- **"Flow"** — Helvetica Neue, weight 700 (bold), tracking −0.04em
- Fallback stack: `'Helvetica Neue', Helvetica, Arial, sans-serif`

The app UI uses **Plus Jakarta Sans** (body) + **Instrument Serif** (display/headings).

## Clear space

Maintain a minimum clear space of **half the icon height** on all sides of the logo.
Never place the logo on a busy background, rotate it, change the colours, or separate "Rent" from "Flow".

## Quick use — add favicon to HTML

```html
<link rel="icon" type="image/svg+xml" href="/rentflow-favicon-32.svg">
```
